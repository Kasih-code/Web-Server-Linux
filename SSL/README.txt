Folder SSL berisi konfigurasi terkait pengaturan HTTPS pada web 
server menggunakan Nginx.

File `myweb.local` merupakan konfigurasi virtual host Nginx yang 
menunjukkan penggunaan SSL/TLS.

File certificate (.crt) dan private key (.key) tidak disertakan dalam
repository karena merupakan file sistem yang disimpan di server
(/etc/ssl) dan bersifat environment-specific.
